This repo serves as the repo for Alex and Dave's Stock trading project under Avion School.


#### Version

```
  Ruby 3.0.0
  Rails 6.1
  NodeJS 14.16.0
  Yarn 1.22.5
```
#### Setup

```
  $ bundle install
  $ rake db:setup
```

#### Starting the application

```
  $ rails server
```

#### How to run the test suite

```
  $ rspec
```

#### How to run the Ruby Linter

```
  $ rubocop
```
