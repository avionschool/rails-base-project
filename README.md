# Avion LMS
A Learning Management System specifically designed for the needs of Avion School

## Phase 1 (MVP): Pre-Course Code Checker

### Models:
- Student
- Admin
- Batch
- Lesson
- Submission
- BatchLesson
- Resources

### Features:
- Student Login and Dashboard
- Sign-up Using GitHub
- Resources
- Built-in Code Checks
- Backup Database

### Integrations:
- GitHub API
- Airtable API

## Phase 2 (For Future Development): Complete LMS

### Models:
- Instructor
- Class
- Skill
- Appointment

### Features:
- Push Notifications
- Downloadable Student Records
- 1:1 Appointments
- Class Calendar Reminders
- Student Attendance
- Skills Tracker
- Mailers

### Integrations:
- Google Calendar API
- Zoom API
